State Park Vegetation Map (In Progress)

This map was created for the final Practicum course of UW-Madison's GIS Development Master's program. The project involed my independent proposal and creation of a web application of my choosing. This map was created to allow users to find information on plant species which may be present in state parks using spatial information on natural communities.

The Interface


A slippy map of Michigan with boundaries of all state parks is shown to the user. Clicking on polygon will populate the side panel with a list of species


Opening each panel will show information about the species and allow the user to search for parks containing that specific species.

How It Works

While the display and user functions are simple, there are complex functions occuring in the script. Selecting each park polygon actually selects multiple hidden polygons which intersect, these are the natural community shapefiles. These natural communities and their associated species come from the Michigan Natural Features Inventory . Each of the natural communities has a many-to-many relationship with a table of species. The script loops through each selected community type and outputs the table information to be placed in the side panel. This setup allows for a function to be saved in each button in the species description to backsearch for parks which intersect natural communities
