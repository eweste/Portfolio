Automated Raster Analysis

This project was created for the Capstone course GEO 777 at UW-Madison. This map analyzes hypothetical data of nitrogen concentration levels and cancer rates.
The full report can be read here. Multiple analyses were performed at once in ArcPy to populated pre-structured ArcPRO layout allowing users to produce multiple maps by varying parameters.

Background


This program was created by taking (hypothetical) randomly sampled nitrate concentrations in point layers and county level data on cancer rates in Wisconsin to find if there was statistical significance between the hypothesis that higher nitrate rates in water cause increases in cancer. The program was written in Python utilizing the ArcPy functions of ESRI ArcPRO.

The Program

Using ArcPy with a tkinter interface, the point layer of nitrate concentration is converted to a raster using inverse distance weighting, with the k value being the input value.

This raster is then averaged over the county shapefiles and has its value joined to the cancer rate shapefiles. This requires multiple background analysis steps and has built in assurances the analyses succeed. This gives the average concentration calculated through IDW for each cancer rate and can then undergo Linear Regression Analysis.

The linear regression returns residuals which show where each feature deviates from the hypothesis that nitrate levels are related to cancer rates.
